# SqlAlchemy Tutorial

**Configuring the environment**

`
pip install virtualenv  
python -m venv alchemy_tuto  
cd alchemy_tuto  
Scripts\activate  
pip install sqlalchemy==1.2.9  
git clone https://github.com/danilobatistaqueiroz/sqlalchemy_basics.git .
`
